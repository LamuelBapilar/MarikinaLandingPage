
# 📘 Marikina Landing Page

---

## 📌 Project Overview

The Marikina Landing Page is a static website that presents information about the City of Marikina. It is designed as a single-page website where all content is displayed.

The project focuses on clean layout, basic navigation, and visual presentation of the city.

Website Link: https://lamuelbapilar.github.io/MarikinaLandingPage/

---

## 🎯 Objectives

- To design a simple and clean landing page  
- To showcase information about Marikina  
- To practice frontend development using HTML and CSS  
- To create a user-friendly interface  

---

## 🛠️ Technologies Used

- HTML5 – structure of the page  
- CSS3 – styling and layout  
- JavaScript – for interactivity  

---

```bash

📍 Landing page design (single-page layout)

🚀 How to Run the Project

1. Clone the repository: git clone https://github.com/LamuelBapilar/MarikinaLandingPage.git
2. Open the project in VS Code
3. Locate index.html
4. Run using Live Server
5. Right click index.html
6. Click Open with Live Server
